---
title: Guia de productos
description: Como buscar, registrar, editar y configurar conversiones de productos.
---

## Ruta

Abrir **Inventario > Productos**.

## Para que sirve

La pantalla **Productos** administra el catálogo farmacéutico. Desde aquí se consultan productos existentes, se registran nuevos productos y se configuran conversiones de presentaciones comerciales hacia la unidad base.

## Buscar un producto

1. Entrar a **Inventario > Productos**.
2. Ubicar la seccion **Lista de productos**.
3. Escribir en **Buscar producto** el código, nombre comercial, nombre genérico o código de barras.
4. Revisar los resultados en la tabla.

## Registrar un producto

1. Entrar a **Inventario > Productos**.
2. En la seccion **Nuevo producto**, revisar **Código interno**. Si queda vacío, el sistema lo asigna al guardar.
3. Completar **Código de barras** si existe.
4. Completar **Nombre comercial**.
5. Seleccionar **Categoría**.
6. Seleccionar **Unidad base**.
7. Seleccionar **Proveedor**.
8. Seleccionar **Tipo**.
9. Escribir **Precio de venta**.
10. Marcar o desmarcar las opciones **Inventariable**, **Exige lote**, **Exige vencimiento** y **Requiere receta** según corresponda.
11. Presionar **Guardar**.

Al guardar, la unidad base queda configurada automaticamente con factor `1`. Esto permite usar el producto en compras sin una configuracion adicional.

## Editar un producto

1. Buscar el producto en la tabla.
2. Presionar **Editar** en la fila del producto.
3. Modificar los campos necesarios en el formulario lateral.
4. Presionar **Guardar**.
5. Para volver al modo de alta, presionar **Nuevo**.

## Configurar conversiones

1. Buscar el producto en la tabla.
2. Presionar **Editar**.
3. Ir a la seccion **Conversiones**.
4. Revisar la unidad base, que debe mantener factor `1`.
5. Presionar **Agregar** para incluir otra presentación comercial.
6. Seleccionar la unidad.
7. Escribir el factor de conversión hacia la unidad base.
8. Presionar **Actualizar**.

La unidad base no debe modificarse para representar cajas, blisters o frascos. Esas presentaciones se agregan como conversiones adicionales.

## Controles visibles

- Campo **Buscar producto**.
- Boton **Editar** en cada fila.
- Formulario **Nuevo producto** o **Editar producto**.
- Selector **Proveedor**.
- Boton **Guardar**.
- Boton **Nuevo**.
- Seccion **Conversiones**.
- Boton **Agregar**.
- Boton **Actualizar**.

## Resultado esperado

El producto queda disponible para compras y consultas con un proveedor único asociado. La unidad base ya queda lista para recibir inventario. Si se configuran conversiones, podrá usarse una presentación comercial distinta de la unidad base al registrar compras de ese proveedor.

## Consejo

Antes de registrar productos, crear primero proveedores, unidades y categorias. Para productos inventariables de farmacia, mantener activadas las opciones de lote y vencimiento salvo que exista una razon operativa clara para no hacerlo.
