---
title: Guia de unidades y categorias
description: Como crear unidades de medida y categorias de producto.
---

## Ruta

Abrir **Inventario > Unidades y conversiones**.

## Para que sirve

Esta pantalla prepara los datos base que luego se usan al registrar productos y compras. Las unidades permiten controlar presentaciones como tableta, caja, blister o frasco. Las categorias agrupan productos como medicamentos, insumos o productos relacionados con farmacia.

## Crear una unidad

1. Entrar a **Inventario > Unidades y conversiones**.
2. En la seccion **Nueva unidad**, completar **Nombre**.
3. Completar **Abreviatura**.
4. Presionar **Guardar unidad**.
5. Confirmar que la unidad aparezca en la tabla **Unidades de medida**.

## Crear una categoria

1. Entrar a **Inventario > Unidades y conversiones**.
2. En la seccion **Nueva categoría**, completar **Nombre**.
3. Completar **Descripción** si se necesita aclarar el uso de la categoria.
4. Presionar **Guardar categoría**.
5. Confirmar que la categoria aparezca en la tabla **Categorías de producto**.

## Controles visibles

- Tabla **Unidades de medida**.
- Tabla **Categorías de producto**.
- Campo **Nombre** para unidad.
- Campo **Abreviatura** para unidad.
- Boton **Guardar unidad**.
- Campo **Nombre** para categoria.
- Campo **Descripción** para categoria.
- Boton **Guardar categoría**.

## Resultado esperado

Despues de guardar, las unidades y categorias quedan disponibles para registrar productos. Si no existen unidades o categorias, el registro de productos queda limitado porque no hay base para definir unidad base ni clasificacion.

## Consejo

Usar abreviaturas breves y claras, como `tab`, `caja`, `blis` o `frasco`. Evitar abreviaturas ambiguas porque luego apareceran en tablas y formularios de compra.
