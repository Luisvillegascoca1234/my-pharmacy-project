title: Primeros pasos
description: Acceso inicial, navegacion principal y reconocimiento de la pantalla.
---

Esta seccion orienta al usuario en el primer contacto con el sistema. Antes de registrar compras, productos o proveedores, conviene reconocer la forma general de navegar y ubicar las acciones principales.

## Iniciar sesion

1. Abrir la aplicacion del sistema POS.
2. En la pantalla **Iniciar sesión**, revisar los campos **Correo electrónico** y **Contraseña**.
3. Escribir las credenciales asignadas.
4. Presionar el boton **Iniciar sesión**.
5. Si el acceso es correcto, el sistema muestra el **Dashboard operativo**.

## Reconocer el panel principal

Al ingresar, la pantalla se divide en tres zonas principales:

- **Barra lateral:** contiene los grupos de navegacion.
- **Encabezado superior:** muestra el modulo actual, el rol del usuario y accesos rapidos.
- **Area de trabajo:** muestra tablas, formularios, metricas o mensajes del modulo seleccionado.

## Usar la barra lateral

La barra lateral agrupa los modulos por area:

- **Inicio:** Dashboard.
- **Operación:** Punto de venta, caja, devoluciones y alertas.
- **Inventario:** productos, unidades, lotes, movimientos y ajustes.
- **Compras:** proveedores y compras.
- **Facturación:** facturas SIAT y configuracion SIAT.
- **Análisis:** reportes, exportaciones CSV y auditoria.
- **Administración:** usuarios, roles y configuracion.

Para abrir una pantalla, hacer clic en el nombre del modulo dentro de su grupo. Por ejemplo, para registrar un proveedor se usa **Compras > Proveedores**.

## Accesos del encabezado

En la parte superior de la pantalla aparecen acciones generales:

- **Alternar navegación:** muestra u oculta la barra lateral.
- **Cambiar tema:** modifica la apariencia visual.
- **Buscar en el sistema:** acceso de busqueda general cuando esté disponible.
- **Alertas operativas:** acceso rapido a observaciones relevantes.
- **Menu de usuario:** muestra el usuario conectado y el rol actual.

## Interpretar el Dashboard

El **Dashboard operativo** muestra tarjetas de resumen como ventas de hoy, stock crítico, compras abiertas y facturas observadas. Tambien muestra el estado general del sistema y alertas iniciales. Esta pantalla sirve para orientar la operacion diaria antes de entrar a un modulo especifico.

## Recomendacion inicial

El orden recomendado para preparar una farmacia nueva es:

1. Crear unidades y categorias.
2. Registrar productos.
3. Registrar proveedores.
4. Crear compras en borrador.
5. Recibir compras para generar inventario.
6. Consultar alertas, reportes y movimientos conforme avance la operacion.
