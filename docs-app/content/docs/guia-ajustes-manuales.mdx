---
title: Guia de ajustes manuales
description: Como corregir stock contado sobre una capa existente con motivo obligatorio.
---

## Ruta

Abrir **Inventario > Ajustes manuales**.

## Para que sirve

La pantalla **Ajustes manuales** permite corregir una capa de inventario existente cuando el stock físico contado no coincide con el saldo del sistema. El usuario escribe el stock final contado; el sistema calcula la diferencia y registra el movimiento.

## Registrar un ajuste

1. Entrar a **Inventario > Ajustes manuales**.
2. Seleccionar **Producto**.
3. Seleccionar **Capa/lote**.
4. Revisar el disponible actual mostrado debajo de **Stock final contado**.
5. Escribir el stock físico real en **Stock final contado**.
6. Escribir un **Motivo** claro.
7. Revisar la **Vista previa** para confirmar la diferencia.
8. Presionar **Registrar ajuste**.

## Interpretar la vista previa

- **Actual:** saldo disponible antes del ajuste.
- **Diferencia:** cantidad que se sumará o restará.
- **Producto y lote:** capa exacta que será corregida.
- **Proveedor:** origen asociado al lote.

## Reglas importantes

- El ajuste no crea productos nuevos.
- El ajuste se aplica sobre una capa existente.
- El motivo es obligatorio.
- El stock final contado no puede ser negativo.
- La operación genera un movimiento de inventario y evidencia administrativa.

## Resultado esperado

El saldo del lote queda corregido y el kardex muestra un movimiento de **Ajuste manual** con cantidad positiva o negativa según corresponda.

## Consejo

Usar motivos verificables, por ejemplo **conteo físico de estantería**, **merma documentada** o **corrección de recepción**. Evitar motivos ambiguos como "ajuste" o "error".
