La informacion esta separada en primeros pasos, guias de uso, conceptos y consejos para que el usuario pueda consultar lo que necesita sin mezclar teoria con instrucciones de pantalla.

## Flujo recomendado de aprendizaje

1. Leer **Primeros pasos** para entender login, navegacion y pantalla principal.
2. Revisar **Guias de uso** para operar unidades, productos, proveedores, compras y usuarios.
3. Consultar **Conceptos** cuando aparezcan terminos como lote, FEFO, kardex, unidad base o SIAT.
4. Usar **Consejos operativos** como lista de buenas practicas antes de registrar informacion sensible.

## Estado de la guia

Las guias fueron redactadas a partir de la interfaz actual. Algunas pantallas, como punto de venta, caja, alertas, facturacion, reportes y auditoria, aparecen como modulos iniciales; por eso se documenta su navegacion y proposito, pero no se inventan acciones que todavia no se muestran en la pantalla.
