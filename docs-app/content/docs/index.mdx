## Flujo recomendado de aprendizaje

1. Leer **Primeros pasos** para entender login, navegacion y pantalla principal.
2. Revisar **Guias de uso** para operar unidades, productos, proveedores, compras y usuarios.
3. Consultar **Conceptos** cuando aparezcan terminos como lote, FEFO, kardex, unidad base, comprobante interno, devolucion administrativa o auditoria.
4. Usar **Consejos operativos** como lista de buenas practicas antes de registrar informacion sensible.

## Estado de la guia

Las guias fueron redactadas a partir de la interfaz actual. Ya existen flujos operativos para catalogos, proveedores, compras recibidas, stock por lote, movimientos, ajustes manuales, alertas de inventario, caja, punto de venta, carritos pendientes, ventas y anulaciones, supervision POS, comprobantes internos, devoluciones administrativas, reportes, exportaciones CSV y auditoria.

El cierre administrativo V1 se documenta como capacidad operativa con limites claros: no emite factura fiscal real, no integra SIAT real, no habilita QR fiscal, no permite devoluciones parciales y no reabre cajas cerradas.
