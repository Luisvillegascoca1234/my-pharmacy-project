---
title: Guia de alertas operativas
description: Como revisar stock critico, agotados y vencimientos desde datos reales.
---

## Ruta

Abrir **Operación > Alertas**.

## Para que sirve

La pantalla **Alertas** resume condiciones que requieren atención operativa. En esta etapa muestra alertas calculadas desde inventario real: stock bajo, productos agotados, lotes próximos a vencer y lotes vencidos.

## Revisar alertas

1. Entrar a **Operación > Alertas**.
2. Revisar los contadores **Críticas**, **Preventivas** y **Generadas**.
3. En **Panel de alertas**, revisar cada fila.
4. Identificar **Prioridad**, **Producto**, **Alerta**, **Lote**, **Vencimiento** y **Disponible**.
5. Presionar **Actualizar** si se quiere recargar la información visible.

## Prioridades

- **Crítica:** requiere atención inmediata, como producto agotado o lote vencido.
- **Preventiva:** conviene revisarla antes de que afecte la operación, como stock bajo o lote próximo a vencer.

## Acciones recomendadas

- Para **Agotado**, revisar compras pendientes o planificar reposición.
- Para **Stock bajo**, anticipar pedido al proveedor.
- Para **Próximo a vencer**, priorizar rotación FEFO o revisar devolución acordada.
- Para **Vencido**, evitar venta y definir tratamiento administrativo.

## Resultado esperado

El usuario puede priorizar tareas sin revisar producto por producto. Las alertas deben señalar qué revisar y por qué.

## Consejo

Las alertas ayudan a decidir, pero no reemplazan el criterio farmacéutico. Ante un lote vencido o una merma, documentar la acción correspondiente y revisar el kardex.
