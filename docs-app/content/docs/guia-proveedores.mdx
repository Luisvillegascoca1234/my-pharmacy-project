title: Guia de proveedores
description: Como buscar, crear, editar y cambiar el estado de proveedores.
---

## Ruta

Abrir **Compras > Proveedores**.

## Para que sirve

La pantalla **Proveedores** administra el padrón comercial usado en compras y recepciones. Un proveedor activo puede seleccionarse para nuevas compras; un proveedor inactivo conserva historial pero deja de estar disponible para nuevas operaciones.

## Buscar proveedores

1. Entrar a **Compras > Proveedores**.
2. Escribir en **Buscar proveedor** la razon social, NIT o contacto.
3. Usar el filtro **Estado** para ver todos, activos o inactivos.
4. Revisar la tabla de resultados.
5. Usar **Anterior** o **Siguiente** si existen varias paginas.

## Crear un proveedor

1. Entrar a **Compras > Proveedores**.
2. Presionar **Nuevo proveedor**.
3. Completar **Razón social**.
4. Completar **NIT** si corresponde. Puede quedar vacio.
5. Completar **Teléfono**, **Contacto** y **Dirección** si se dispone de esos datos.
6. Revisar el bloque **Estado operativo**.
7. Si el proveedor debe iniciar inactivo, presionar **Crear como inactivo** y confirmar.
8. Presionar **Guardar**.

## Editar un proveedor

1. Abrir **Compras > Proveedores**.
2. Buscar el proveedor.
3. Abrir el detalle desde la accion disponible en la tabla.
4. Modificar los datos comerciales.
5. Presionar **Guardar**.
6. Si no se desea conservar lo editado, presionar **Descartar cambios** y confirmar con **Descartar**.

## Activar o desactivar un proveedor

1. Abrir el detalle del proveedor.
2. Revisar la tarjeta **Estado operativo**.
3. Presionar **Desactivar proveedor** o **Activar proveedor**, segun el estado actual.
4. Leer la confirmacion.
5. Presionar **Confirmar**.

## Controles visibles

- Link **Nuevo proveedor**.
- Campo **Buscar proveedor**.
- Filtro **Estado**.
- Boton **Guardar**.
- Boton **Descartar cambios**.
- Boton **Crear como inactivo**, **Desactivar proveedor** o **Activar proveedor**.
- Link **Volver**.

## Resultado esperado

El proveedor queda registrado y disponible para compras si está activo. Si se desactiva, no se elimina su historial ni las compras asociadas.

## Consejo

Registrar el NIT cuando exista, pero no bloquear el alta si el proveedor aun no entregó ese dato. La razon social debe ser suficientemente clara para reconocerlo al crear compras.
